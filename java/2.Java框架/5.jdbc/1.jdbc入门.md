# JDBC入门
## JDBC概念
1. 概念：**Java DataBase Connectivity**  
Java 数据库连接， Java语言操作数据库

* JDBC本质：其实是**官方（sun公司）定义的一套操作所有关系型数据库的规则，即接口**。各个数据库厂商去实现这套接口，提供数据库驱动jar包。我们可以使用这套接口（JDBC）编程，**真正执行的代码是驱动jar包中的实现类**。
