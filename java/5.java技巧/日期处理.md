# java在当前系统时间加 N 天

```java
    /**
     * 根据日期获取20天后的日期（过期日期）
     * @param date
     * @return date
     */
    public static Date getExpiredDay(Date date) {
        Calendar calendar = Calendar.getInstance();
        calendar.setTime(date);
        calendar.add(Calendar.DAY_OF_MONTH, +20);//+20今天的时间加一天
        date = calendar.getTime();
        return date;
    }

```

## 判断是否过期
```java
    /** 判断是否过期
     * @param today
     * @param dateofproduct
     * @return
     */
    public static boolean isExpired(Date today,Date dateofproduct){
        if(today.before(getExpiredDay(dateofproduct))){
            return false;
        }else {
            return true;
        }
    }
```
## 获取当天的日期
1. Date类型转换成String类型
```java
public static void main(String[] args)  {
        SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
        Date date = new Date();
        String dateString = sdf.format(date);
        System.out.println(dateString);
    }
```
* 结果：2019-06-14

2. String类型转Date
```java
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd");
Date today = new Date();
dateinfo = sdf.format(today)//获取现在的日期
Date dateinfoformat = sdf.parse(dateinfo) // String格式转换为Date格式
```
