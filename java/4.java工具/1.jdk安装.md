# jdk安装

1. 安装选择默认路径  
2.配置环境变量  

* （1）新建JAVA_HOME变量  
变量名：JAVA_HOME   
变量值：填写JDK安装目录（我的是 D:\Program Files (x86)\Java\jdk1.8.0_181）  

* （2）找到Path变量进行编辑，没有的话就新建一个  
变量名：Path  
变量值： ;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;  

* （3）新建CLASSPATH变量
变量名：CLASSPATH  
变量值： .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar  
//可以直接复制这句话到变量值那一栏，注意前面有个“.”不要漏掉
